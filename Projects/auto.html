Purchased

Cube Orange

Primary flight controller for the build, with ArduPilot support and a modular carrier-based design.

Here 3 GPS

GNSS and compass module used for positioning, navigation, and heading data on the aircraft.

RadioMaster Boxer

Main handheld radio controller for piloting and configuring the drone from the ground.

RadioMaster RP1 Receiver

ExpressLRS 2.4 GHz nano receiver that links the aircraft to the radio controller.

Holybro SiK Telemetry Radio V3

Telemetry radio pair for long-range data, mission updates, and live communication with the vehicle.

2250mAh LiPo Battery

Main flight battery for the quadcopter, providing the onboard power needed for motors and avionics.

CubePilot Power Brick Mini

Compact power module that conditions battery input and provides voltage and current monitoring to the flight controller.

F450 Frame Kit

Quadcopter frame platform that provides the arms, center plates, and mounting structure for the build.

Racerstar BR2312 960KV Motors

Brushless motors for propulsion, sized for multi-rotor use on a 350 to 400 class style frame.
